# Phase 3：実用機能（並び替え / フィルタ / 検索 / 重要度）

## 目的
MVPを「使える」状態に引き上げる。

## やること
- フィルタ条件を実装する
- 並び順を実装する
- 重要度 / ステータスを実装する
- 期限ロジック（今日 / 明日など）を実装する

## まとめるもの
- 仕様の決定事項（例：優先度の値域、並び順の優先）
- 例外ケース（例：期限なしをどう扱うか）

## 作業チェックリスト
- [ ] フィルタの対象項目と条件を定義した
- [ ] 並び順の優先ルールを定義した
- [ ] 重要度とステータスの値域を固定した
- [ ] 期限判定ルール（今日 / 明日 / 期限切れ）を定義した
- [ ] 期限なしデータの表示・並び順を定義した
- [ ] 条件の組み合わせ時の動作（例：検索 + フィルタ）を確認した

## 完了条件
- ユーザーが迷わず目的のデータに辿り着けること。
